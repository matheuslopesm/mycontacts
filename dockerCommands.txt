Instalar a imagem:
docker pull postgres

Listas todas imagens instaladas na m√°quina:
docker image ls

Criar container:
docker run --name (NOMEDOCONTAINER) POSTGRES_USER=NOMEDOUSER -e POSTGRE_PASSWORD=NOMEDOPASSWORD -p 5432:5432 -d postgres

Ver se tem container criado e rodando:
docker container ls
ou
docker ps

Parar de executar o container:
docker stop (NOME DO CONTAINER)
ou
docker stop (CONTAINER ID)

Ver container parado:
docker container ls -a
ou
docker ps -a

Startar o container:
docker start (NOMEDOCONTAINER)

Apagar imagem:
docker rmi (IMAGEID)

Executar o container:
docker exec -it (NOMEDOCONTAINER) bash

Executar o postgres no container:
psql -U (NOMEDOUSER)
